# Information Security Policies

## Acceptable Use Policy
All employees must use company IT resources responsibly. Personal use is permitted within reason but must not interfere with work duties or violate any policies. Prohibited activities include: accessing inappropriate content, cryptocurrency mining, unauthorized data sharing, and using company resources for personal business ventures.

## Data Classification
- **Public**: Marketing materials, press releases, public website content.
- **Internal**: Internal communications, meeting notes, project plans.
- **Confidential**: Customer data, financial records, HR information, source code.
- **Restricted**: Encryption keys, security audit results, incident response plans, executive communications.

Confidential and Restricted data must be encrypted at rest and in transit. Never share Restricted data via email without encryption.

## Incident Reporting
If you suspect a security incident (phishing email, unauthorized access, data leak, malware), report it immediately:
1. Email: security@company.com
2. Phone: ext. 9111 (24/7 Security Operations Center)
3. Slack: #security-incidents channel

Do NOT attempt to investigate or contain the incident yourself. Preserve evidence by not deleting emails or files. Disconnect from network if you suspect active malware.

## Remote Work Security
- Always use the VPN when accessing company resources remotely.
- Lock your screen when stepping away (Win+L or Cmd+Control+Q).
- Do not use public Wi-Fi without VPN â€” use your phone hotspot if VPN is unavailable.
- Ensure your home router firmware is up to date and uses WPA3 or WPA2.
- Company data must not be stored on personal devices without MDM enrollment.

## Password Policy
- Minimum 12 characters
- Must include: uppercase, lowercase, number, special character
- Changed every 90 days
- Cannot reuse last 12 passwords
- Account locks after 5 failed attempts (30-minute lockout)
- Use the company password manager (1Password) for all work accounts
- Never share passwords or store them in plaintext

## USB and Removable Media
USB ports on company laptops are restricted by default. To use a USB drive for legitimate business purposes, submit an exception request through the IT Portal. All USB drives must be company-issued and encrypted. Personal USB drives are never permitted on company devices.
